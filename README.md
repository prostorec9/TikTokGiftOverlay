# 🎁 TikTok Gift Overlay для Android

Показывает подарки TikTok LIVE **поверх любого приложения** на Android.
Работает даже на рабочем столе, в любом другом приложении, при заблокированном экране.

---

## 📲 Как собрать APK через GitHub (бесплатно, без ПК)

### Шаг 1 — Создай аккаунт GitHub
Зайди на https://github.com и зарегистрируйся (бесплатно).

### Шаг 2 — Создай репозиторий
Нажми **"New repository"**, назови `TikTokGiftOverlay`, нажми **"Create repository"**.

### Шаг 3 — Загрузи файлы
Нажми **"uploading an existing file"** и перетащи ВСЕ файлы из этой папки.
Нажми **"Commit changes"**.

### Шаг 4 — GitHub автоматически соберёт APK
Зайди во вкладку **Actions** → подожди ~3-5 минут пока собирается.

### Шаг 5 — Скачай APK
В Actions → последний запуск → **Artifacts** → скачай `TikTokGiftOverlay-debug`.

---

## 📱 Как установить APK на телефон

1. Скачай APK на телефон
2. Настройки → Безопасность → **"Установка из неизвестных источников"** — включить
3. Открой APK → Установить

---

## ⚙️ Как пользоваться

1. Открой приложение **TikTok Gift Overlay**
2. Выдай разрешение **"Поверх других приложений"** (приложение само попросит)
3. Введи **@username** TikTok стримера чей LIVE ты смотришь
4. Нажми **НАЧАТЬ**
5. Сверни приложение — подарки будут выезжать слева поверх всего!

---

## 🔧 Что делает приложение

- Подключается к TikTok LIVE через WebSocket
- Получает события подарков в реальном времени
- Отображает анимированные карточки слева на экране (как на TikTok)
- Работает поверх ЛЮБОГО приложения и рабочего стола
- Автоматически переподключается при потере связи
- Автозапускается после перезагрузки телефона

---

## 📋 Требования
